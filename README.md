# Akhlaq Kafel — Portfolio

Personal portfolio of **Akhlaq Kafel**, Full-Stack Software Developer based in Jammu & Kashmir, India.

Built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, Lenis, and Three.js.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — development
- `npm run build` — production build
- `npm run start` — serve production build
- `npm run lint` — ESLint

## Content

All personal copy, projects, experience, and links live in `src/data/portfolio.ts`.
Resume: `public/Akhlaq_Kafel_Resume.pdf`.
