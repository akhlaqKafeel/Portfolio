# Akhlaq Kafel — Portfolio

Premium full-stack developer portfolio built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, Lenis, GSAP, and Three.js.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

1. **Portrait** — Already set to `public/portrait.png`. Replace that file to update the hero photo.
2. **Resume** — Replace `public/resume.pdf` with your real resume.
3. **Links** — Update email, LinkedIn, GitHub in `src/data/portfolio.ts`.
4. **Projects** — Update live/GitHub URLs and copy in `src/data/portfolio.ts`.
5. **Testimonials** — Swap placeholder quotes with real ones.

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run start` — serve production build
- `npm run lint` — ESLint
